# Задание №6. Сервер вычислений.

## Цель работы
Приобрести навыки программирования транспортных протоколов (сокетов) на транспортном уровне и клиент-серверных приложений на уровне приложений.

## Задание
Разработать сервер вычислений, который получает запросы на вычисление (например, факториала или арифметического выражения) и возвращает результат клиенту.

## Функциональные требования
- Парсинг и обработка запросов.
- Корректные вычисления.
- Обработка ошибок при неверных данных.
- Реализация простого HTTP-сервера на сокетах.

## Краткое описание реализации
Сервер принимает HTTP GET-запросы с параметрами `factorial` или `expr`.
- Для `factorial` вычисляет факториал числа.
- Для `expr` вычисляет арифметическое выражение с базовой проверкой.
Ошибки в запросах или вычислениях обрабатываются и возвращаются в ответ клиенту.
